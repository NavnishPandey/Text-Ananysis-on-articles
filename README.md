# Text-Analysis-on-articles
 Scraped data from websites and perform analysis and calculate the statistical scores for eg :-
 POSITIVE SCORE,
 NEGATIVE SCORE,
 POLARITY SCORE.
 SUBJECTIVITY SCORE,
 AVERAGE SENTENCE LENGTH,
 PERCENTAGE OF COMPLEX WORDS,
 FOG INDEX,
 PERSONAL PRONOUNS,
 COMPLEX WORD COUNT,
 SYLLABLE PER WORD,
 AVG WORD LENGTH
